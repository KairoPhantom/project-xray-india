# Case research template

## Identity
Project, authority, location, stable IDs, packages/contracts.

## Search log
Record query, portal, filters, date/time, result and archive/hash. A gap is valid only with declared search scope.

## Timeline
Approval, tender, award, amendments, planned/revised completion, commissioning, incident, repair, defect period.

## Claims
For each: atomic text, type, source class, exact passage/page, supporting/contradicting record, reviewer.

## Required records
DPR, tender, bid summary, award, signed contract, amendments, safety/design review, quality tests, measurement/payment certificate, completion, defect terms, incident report, action taken.

## Fairness
Official/contractor response, strongest alternative explanation, and “what evidence does not establish.”
