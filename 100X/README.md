## Core strategic decision
Do **not** build another generic corruption dashboard, generic RAG chatbot, tender search engine or allegation platform. Build a **source-grounded infrastructure evidence compiler**: an auditable workflow that transforms fragmented official records into reviewable claims, gaps, timelines, responses and reproducible evidence packages.
